Coding Art

I've taken quite a few programming classes in college, and as a result I have created some pretty neat pieces of art. All of these were created with javascript and processing. These are all interactive within this page.

2017
A spinning sculpture that reacts to the volume of the room. On mobile, the sculpture spins according to the device's orientation. [Not compatible with Safari. Works best on Chrome.]
(This sketch asks for microphone permission, and only uses the amplitude levels from the mic. Don't worry, I'm not recording you.)

2016
A sword sketch, with colors and blade length generated randomly. Clicking on the sketch reshuffles the swords.

2017
Rock, paper, scissors within their own little ecosystem. When only one type of agent remains, a new generation is born.

2016
A pushed 3D version of the sword sketch following the same rules, using tops instead.
